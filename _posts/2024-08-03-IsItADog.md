[Practical Deep Learning](https://course.fast.ai).  This is a free course designed for people
who want to learn how to apply deep learning and machine learning to partical problems.

if a image is a dog.  You can access my Jupyter Notebook for this model from my [Kaggle.com home page](https://www.kaggle.com/davidphummel).  The Code link contains the notebook called *Computer Vision - Is it a dog?*.
