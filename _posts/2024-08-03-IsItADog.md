# Computer Vision - Is it a Dog?

I have recently started to teach myself how to create new models using the fast.ai course 
[Practical Deep Learning](https://course.fast.ai).  This is a free course designed for people
who want to learn how to apply deep learning and machine learning to partical problems.

I have read chapter 1 and modified the first exercise to create a computer vision model to idenitfy
if a image is a dog.  My Jupyter Notebook [Computer Vision - Is it a dog?](https://www.kaggle.com/davidphummel) contains the
details of my work.  You can access this workbook by selecting the Code link on my Kaggle.com page.

I am currently working on creating a similar model from code which was generated
from ChatGPT to solve the same problem.
